快速上手 #

跟随以下的步骤，快速上手组件库的使用。

Vue 版本 #

vue >= 3

💡 友情提示

因为我们当前组件库是基于Vant 4.x 版本 二次封装的，所以必须确保在项目已经安装和正确使用了vant4，vant 安装使用指南

安装 #

shell
# npm
$ npm install envision-ui -S

# yarn
$ yarn add envision-ui -S

# pnpm 
$ pnpm add envision-ui -S

完整引入 #

在 main.js 文件当中写入以下代码：

js
import { createApp } from 'vue'
import App from './App.vue';
import { Progress } from 'vant';
import 'vant/lib/index.css';
import envsionUI from 'envision-ui'
import 'envision-ui/style.css'


const app = createApp(App);
app.use(envsionUI).use(Progress).mount("#app")

❌ 注意

envision-ui 需要单独导入全局样式

按需加载 #

可以使用手动导入的方式按需加载组件。

js
import { createApp } from 'vue'
import App from './App.vue'
import { Progress } from 'vant'
import 'vant/lib/index.css'
import 'envision-ui/style.css'
import {envProgress,envFixedSideNav,envTreeMenu} from 'envision-ui'

app.use(envProgress).use(envFixedSideNav).use(envTreeMenu)
.use(Progress).mount('#app')
