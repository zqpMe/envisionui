开关 Switch #

用于在打开和关闭状态之间进行切换。

基本用法 #

<template>
  <div class="switch-box">
    <van-switch v-model="checked" />
    <span style="margin: 0 0 0 10px">{{
      checked ? "打开状态" : "关闭状态"
    }}</span>
  </div>
</template>

<script setup>
import { ref } from "vue";
const checked = ref(true);
</script>
<style lang="less" scoped>
.switch-box {
  display: flex;
  align-items: center;
}
</style>

禁用状态 #

禁用开关。

<template>
  <van-switch v-model="checked" disabled />
</template>
  
<script setup>
import { ref } from 'vue';
const checked = ref(true);
</script>

⭐️ 温馨提示

开关处于禁用状态时不可点击

基本布局 #

<template>
  <van-cell-group>
    <van-cell title="标题文字" style="text-align: left">
      <template #right-icon>
        <van-switch v-model="checked" />
      </template>
    </van-cell>
  </van-cell-group>
  <van-cell-group>
    <van-cell title="禁用状态文字标题" style="text-align: left">
      <template #right-icon>
        <van-switch v-model="checked" disabled />
      </template>
    </van-cell>
  </van-cell-group>
</template>

<script setup>
import { ref } from "vue";
const checked = ref(true);
</script>
