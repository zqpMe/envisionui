日期分段选择 DateRangeInput #

适用于选择日期区间

基本用法 #

当设置type为range时可以使用<env-date-range-input/>,
除了以下特别的props属性,同样可以使用vant中日历组件的其他props。

Props #

参数名描述类型默认值
isShow是否显示日历弹窗booleanfalse
title日历标题string''
cancalText取消按钮的文字string'取消'
confirmText确定按钮的文字string'确定'
type选择类型:single 表示选择单个日期，multiple 表示选择多个日期，range 表示选择日期区间string'single'

Slots #

插槽名描述参数
footer自定义底部区域内容-

Events #

事件名描述参数
select点击并选中任意日期时触发value: Date 或 Date[]
open打开弹出层时触发-
close关闭弹出层时触发-
opened打开弹出层且动画结束后触发-
closed关闭弹出层且动画结束后触发-
cancel点击取消文字后触发-
confirm-button击确认按钮后触发-
confirm日期选择完成后触发value: Date 或 Date[]
month-show当某个月份进入可视区域时触发{ date: Date, title: string }
over-range范围选择超过最多可选天数时触发-
click-subtitle点击日历副标题时触发event: MouseEvent

实例方法 #

方法名描述参数返回值
reset将选中的日期重置到指定日期，未传参时会重置到默认日期--
scrollToDate滚动到某个日期date?: DateDate[]
getSelectedDate获取选中的日期-Date
