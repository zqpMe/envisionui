按钮 Button #

按钮用于触发一个操作，如提交表单。

基本用法 #

按钮类型 #

按钮支持 primary、gray、success、default、danger、warning、ghost1、ghost2 八种类型，默认为 default。

禁用功能 #

通过 disabled 属性设置按钮禁用功能。

图标按钮-左右 #

通过style="border: none"， 结合 plain icon 属性设置按钮图标，支持 Icon 组件里的所有图标，也可以传入图标 URL。

文字按钮 #

通过 type 属性可以设置文字按钮。按钮支持 text-danger、text-primary、text-default三种类型。通过 disabled 可以设置禁用状态。

块级按钮 #

通过block 属性可以将按钮的元素类型设置为块级元素。

通栏按钮 #

通过 type 属性可以设置通栏按钮。按钮支持 primary-banner、default-banner、warning-banner、danger-banner、success-banner 五种类型。 通过 disabled 可以设置禁用状态。

按钮组 #

通过在 <env-sticky-button>可以设置按钮组。
通过offset设置扇面偏移角，默认是四分之π，配合默认方向lt,
通过direction控制方向,
分为左上( lt )、上( t )、右上( rt )、右( r )、右下( rb )、下( b )、左下( lb )、左( l )，默认左上( lt ),
通过radius控制扇面半径
