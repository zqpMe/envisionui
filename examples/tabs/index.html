标签页 Tab #

选项卡组件，用于在不同的内容区域之间进行切换。

基本用法 #

通过line-width可以设置底部条宽度，默认单位 px

带边框 #

Tab 支持两种样式风格：line 和card，默认为 line 样式，可以通过 type 属性切换样式风格。

组合 #

Tab 支持line 和card样式风格的组合
