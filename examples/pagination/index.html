Pagination 分页 #

数据量过多时，采用分页的形式将数据分隔，每次只加载一个页面。

样式一 #

需切换到简单模式,将mode 设置为 simple,通过showBorder为true可显示上一页、下一页按钮的边框,

样式二 #

需切换到简单模式,将mode 设置为 simple,通过homePageText和lastPageText可设置首尾页按钮文字

样式三 #

需切换到简单模式,将mode 设置为 multi,

API #

Props #

参数名描述类型默认值
v-model当前页码number-
showBorder是否显示上一页、下一页的边框，当mode为simple时生效booleanfalse
mode显示模式，可选值为 simplestringmulti
pageCount总页数number | string根据页数计算
totalItems总记录数number | string0
itemsPerPage每页记录数number | string10
modelValue当前页码number-
homePageText自定义首页按钮文字string''
lastPageText自定义尾页按钮文字string''

Events #

事件名描述参数
change页码改变时触发-

Slots #

名称描述参数
page自定义页码{ number: number, text: string, active: boolean }-
prev-text自定义上一页按钮文字-
next-text自定义下一页按钮文字-
